# 环境配置



## 配置环境变量

在 ./profile 末尾加：

```
export PATH=/opt/miniforge3:$PATH

# >>> mamba initialize >>>
# !! Contents within this block are managed by 'mamba shell init' !!
export MAMBA_EXE='/opt/miniforge3/bin/mamba';
export MAMBA_ROOT_PREFIX='/opt/miniforge3';
__mamba_setup="$("$MAMBA_EXE" shell hook --shell zsh --root-prefix "$MAMBA_ROOT_PREFIX" 2> /dev/null)"
if [ $? -eq 0 ]; then
    eval "$__mamba_setup"
else
    alias mamba="$MAMBA_EXE"  # Fallback on help from mamba activate
fi
unset __mamba_setup
# <<< mamba initialize <<<

export PATH=/usr/local/cuda-12.8/bin:$PATH
```

 

在终端中：

```
source ~/.profile
```

并且将以上这行加到 .zshrc 中。

 

确保使用的终端是 zsh。

## 下载 demo 数据

```
mkdir datasets
cd datasets
wget https://repo-sam.inria.fr/fungraph/3d-gaussian-splatting/datasets/input/tandt_db.zip
unzip tandt_db.zip
```

## 运行

训练：

```
python train.py -s ./datasets/tandt/train -m ./output/train_model
python render.py -m ./output/train_model # 按照需要渲染图片
```

产生的模型会存在 output 里面，使用 -m 可指定文件夹名称，否则会随机生成

 

训练并评估：（替换成实际模型路径）

```
python train.py -s ./datasets/tandt/train/ --eval
python render.py -m /home/anya/gaussiansplatting/output/e693ff4c-4
python metrics.py -m /home/anya/gaussiansplatting/output/e693ff4c-4
```

 

train.py 常用 Arguments：

```
--iterations 10000
--test_iterations 3000 7000
--save_iterations 3000 7000
```

Viewers 常用功能：

- 在 Camera Point View 窗口中：
    - 第一行切换控制 Mode：
        - FPS： WSADQE 平移，IKJLUO 旋转。
        - Trackball：鼠标左键点击旋转，右键点击平移。
    - 第二行 Snap to closest：移动到最近的输入相机的位置和角度。
    - 最后一行修改平移和旋转速度。
- 在 3D Gaussians 窗口中：
    - 第一行切换 Render Mode：Splats / Initial Points。
- 左上角 Views 中，勾选 Top View，可以查看输入相机位置，初始点云，和目前相机位置。
- 左上角 Capture 中，先 Set export directory...，然后可以导出需要的 Top view / Point View。
